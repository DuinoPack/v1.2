Serial

parseInt()

Description

Looks for the next valid integer in the incoming serial stream. parseInt() inherits from the Stream utility class.

Syntax

Serial.parseInt()

Arduino Mega only:
Serial1.parseInt()
Serial2.parseInt()
Serial3.parseInt()

Parameters

none

Returns

int : the next valid integer
